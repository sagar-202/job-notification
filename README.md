# Job Notification Tracker

A premium SaaS webapp for precision-matched job discovery, built with the KodNest Premium Design System.

## Features

- **Landing Page**: Hero section with clear value proposition
- **Settings**: Configure job preferences (role, location, work mode, experience)
- **Dashboard**: View matched job opportunities
- **Saved**: Bookmark jobs for later review
- **Digest**: Daily job summary delivered at 9AM
- **Proof**: Track implementation progress

## Design System

Built on **KodNest Premium Build System**:
- Calm, intentional, coherent design
- Off-white background (#F7F6F3)
- Deep red accent (#8B0000)
- Serif headings (Crimson Pro)
- Clean sans-serif body (Inter)
- Consistent spacing scale (8, 16, 24, 40, 64px)

## Getting Started

1. Open `app/index.html` in your browser
2. Navigate through the application using the top navigation
3. Configure your preferences in Settings
